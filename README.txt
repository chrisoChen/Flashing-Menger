CPSC 453 cube in Core OpenGL.
Renders a 3-D Menger Sponge with a 'flashing' texture.
Press arrow keys to rotate cube. Enter recursion steps in code to change depth of recursion.

Initially run on CMake via JetBrains Clion.
Instructions to run prjoect with Jetbrains Clion, :
    Open project in Jetbrains Clion, ensure mingw32 is installed and configured in the following path:
        Settings->Build, Execution, Deployment->Toolchains select appropriate Make, C Compiler, and C++ Compiler.
    CMake should initially run to configure files. Press 'Play' Button to start program.
